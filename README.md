# HJtools
The main purpose of this app is to organize the general meetings, goals, thoughts and so on, of the young man organization

## Technologies
* Angular 4
* Ionic 3
* Firebase

## Current features
* List of agendas (Sunday school)
* Create/delete/update agendas
